# createTimeout

Creates a promise that resolves after given timeout.

## Reference

```ts
createTimeout(ms = 0): Promise<void>
```

### Returns

Promise that resolves after given timeout.

## Usage

Using `async/await`

```ts
await createTimeout(1000);

console.log("I'm called after 1 second");
```

Using `Promise.then`

```ts
const controller = new AbortController();

createTimeout(2000).then(() => {
  controller.abort();
});

// An error is thrown when request is aborted
const response = await fetch('https://example.com/', {
  signal: controller.signal,
});
```
